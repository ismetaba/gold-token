# GOLD Token — Sunum

Yönetim kurulu seviyesi sunum. Modern dark-luxury tema, altın aksanlar.

## Açma

```bash
# Direkt
open presentation/index.html

# Veya local server
cd presentation && python3 -m http.server 8000
# → http://localhost:8000
```

## Navigasyon

- **← / →** önceki / sonraki slayt
- **Space / PageDown** sonraki
- **Home / End** ilk / son
- **F** fullscreen
- Dokunmatik: yatay swipe
- Alt orta: nokta navigasyon

## Slaytlar

1. Cover — GOLD monogramı + temel rakamlar
2. Misyon — vizyon + 4 istatistik
3. Tasarım İlkeleri — 4 direk
4. Yetki Alanları — TR · CH · AE · LI
5. Sistem Mimarisi — C4 konteyner
6. Rekabet — PAXG / XAUT karşılaştırma
7. Teknoloji Yığını
8. Yol Haritası — 4 faz
9. Kapanış
